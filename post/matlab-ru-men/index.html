Matlab入门

2021-08-26 3 min read

一、随机数矩阵

  1. 标准正态分布
    randn(m,n)
  2. 均值为μ，标准差为σ的正态分布
    normrnd(μ,σ,m,n)
  3. 均值为μ的指数分布
    exprnd(u,m,n)
  4. 参数为λ的泊松分布
    poissrnd(λ,m,n)
  5. 在区间[a,b]之间均匀分布
    unifrnd(a,b,m,n)
  6. 在区间[a,b]之间均匀分布的整数
    randi([a,b],[m,n])

二、全排列

  1. 随机产生1到8的一个全排列
    randperm(8)
  2. 产生3到5的所有全排列
    perms([3,5])

三、求极限

syms x a
eqn=(1+a/x)^x;
l=limit(eqn,x,inf)

四、求导数

syms x
eqn=log((x+2)/(1-x));
d3y=diff(eqn,x,3)

五、求极值

syms x
y=x^3+6*x^2+8*x-1;
dy=diff(y);
xx=solve(dy)    % 解导数方程得到原函数极值点
fplot(y);

六、求积分

1.不定积分

syms x
eqn=1/(1+sqrt(1-x^2));
I=int(eqn);

2.定积分

  1. 一重定积分
y=@(x)1/(x*(x^2‐3*x+2)^(1/3));
I=integral(y,2,inf)
  1. 二重定积分

先将二重定积分化为累次积分

f=@(x,y)sqrt(1-x.^2-y.^2);
ymax=@(x)sqrt(x-x.^2);  ymin=@(x)-sqrt(x-x.^2);
I=integral2(f,0,1,ymin,ymax)

七、级数求和

syms n
f=(2*n-1)/2^n;
s=symsum(f,n,1,inf)

八、mat文件的存储和加载

a=rand(3,4);
b=poissrnd(1,2,3);  % 均值为1的泊松分布
% 1.存储
save data_test.mat a b;     % 将数据a、b存入文件data_test.mat中
% 2.加载
load('data_test.mat');

九、txt文件的读写

% 1.写文件
b=randi([0,10],3); %生成3阶随机整数矩阵
writematrix(b,'data_test.txt');
% 2.读文件
b=readmatrix('data_test.txt');

十、Excel文件的读写

% 1.写文件
%{
    把一个5×10矩阵写到文件data_test.xlsx，Sheet2，B2开始的域中
%}
a=rand(5,10); 
writematrix(a, 'data_test.xlsx', 'Sheet', 2, 'Range', 'B2');
% 2.读文件
%{
    将data_test.xlsx，Sheet2 “C3：F6”中的数据赋给b
%}
b=readmatrix('data_test.xlsx','Sheet',2,'Range','C3:F6')
