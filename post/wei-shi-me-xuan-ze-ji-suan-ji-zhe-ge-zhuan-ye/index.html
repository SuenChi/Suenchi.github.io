为什么选择计算机这个专业

2021-08-30 2 min read

零、目前情况

01.客观情况概述

  1. 大三上学期开学第一天
  2. 跟随学校的进度学习了：
  • C
  • C++
  • python
  • Java
  • 数据库
  • 数据结构
  • 数字逻辑
  • 计算机组成原理
  • 汇编语言
  • 离散数学
  • 算法设计与分析
  1. 参加过上学期的“蓝桥杯”程序竞赛，获得省二等奖（花了300块报名费，亏死）
  2. 再过一周将参加数学建模竞赛，队内担任编程手，准备不充分

02.主观情况概述

  1. 常常放纵自己刷短视频
  2. 总是漫无目地在电脑面前寻找刺激
  3. 总是习惯性地对周围事物不求甚解

03.为什么选择计算机这个专业？

  1. 讨厌原来的专业，觉得学不到什么有用的东西，将来也不想从事相关的行业
  2. 从小就对电脑感兴趣，算得上家里的电脑小能手
  3. 学好本专业的先决条件很容易满足，只需要一个笔记本电脑
  4. 只需要在电脑上敲几行代码便能实现各种各样的功能，不需要其他金钱的投入，我觉得很神奇
  5. 看到自己敲好的代码成功运行后，非常有成就感

04.为什么讨厌这个专业？

  1. 计算机行业加班严重
  2. 掉头发
  3. 需要不断学习
  4. 老员工没有新员工有竞争力
  5. 感觉自己像刚成熟的柠檬一样，被榨干后，便被丢弃
  6. 一点也不觉得我在为社会创造价值，只是在为少数人赚钱服务
  7. 这个行业体系很成熟，从零基础到入职，好像在打别人为你设置的游戏关卡一般，我讨厌这种感觉
  8. 根本的一点还是觉得我没有创造任何社会价值，没有给国家和人民作出贡献
